# Project 1: IT Support Home Lab

## Objective
To build a basic IT support home lab that demonstrates system setup, user management, troubleshooting, and basic networking skills.

## Tools Used
- VirtualBox / VMware
- Windows 10
- Ubuntu Linux
- Command Line (CMD / Terminal)

## Lab Setup
- Created a Windows 10 virtual machine to simulate an end-user workstation
- Created an Ubuntu Linux virtual machine to simulate an admin or support system

## Tasks Performed
- Installed operating systems
- Created and managed user accounts
- Installed and removed applications
- Simulated common IT support issues
- Performed system troubleshooting
- Practiced basic networking commands

## Issues Simulated & Resolutions
- User unable to log in → password reset
- Slow system performance → disk cleanup and startup optimization
- Network connectivity issue → IP configuration and ping tests

## What I Learned
- How IT support environments are structured
- Importance of documentation in troubleshooting
- Basics of user support and system maintenance
