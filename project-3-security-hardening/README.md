# Project 3: Basic System Security & Hardening

## Objective
To demonstrate foundational cybersecurity knowledge through basic system hardening, access control, and threat mitigation techniques relevant to IT support and entry-level security roles.

## Security Concepts Applied
- Principle of Least Privilege
- Strong password policies
- System updates and patch management
- Firewall configuration
- User access control
- Basic threat identification

## Environment
- Windows 10
- Ubuntu Linux

## Security Tasks Performed

### 1. User Account Security
- Created standard user accounts with limited privileges
- Removed unnecessary administrative access
- Enforced strong password practices

### 2. System Hardening
- Enabled automatic system updates
- Disabled unnecessary startup applications
- Reviewed and secured system services

### 3. Firewall Configuration
- Verified Windows Defender Firewall status
- Enabled firewall rules
- Confirmed active firewall protection

### 4. Basic Threat Awareness
- Identified phishing email indicators
- Recognized common malware behaviors
- Applied basic mitigation steps

### 5. Access Control & Auditing
- Reviewed login activity
- Checked system logs for unusual behavior
- Documented security findings
